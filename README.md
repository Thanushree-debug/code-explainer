# 💻 AI Code Explainer

A beginner-friendly AI-powered app that explains code in simple English, detects bugs, suggests improvements, and analyzes time complexity.

## 🚀 Live Demo
[Click here to try the app](https://code-explainer-eygesjqlguundpvfq7kcne.streamlit.app)

## ✨ Features
- 📖 Step-by-step code explanation in simple English
- 🐛 Bug detection and fixes
- ⚡ Time complexity analysis (Big-O)
- 💡 Code improvement suggestions
- 🌐 Supports Python, C, Java, JavaScript
- 💬 Chat-style interface

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3.3 70B)

## ⚙️ How to Run Locally

1. Clone the repo
```
git clone https://github.com/Thanushree-debug/code-explainer.git
```

2. Install dependencies
```
pip install -r requirements.txt
```

3. Add your Groq API key in `.streamlit/secrets.toml`
```
GROQ_API_KEY = "your_key_here"
```

4. Run the app
```
streamlit run app.py
```
